# Tripleten web_project_homeland

## Descrição

O projeto Pátria é um site responsivo projetado para funcionar adequadamente entre dispositivos de 320px a 1280px de largura. O projeto utiliza metodologia BEM e BEM Flat para melhor organização do código e arquivos.

## Tecnologias e Técnicas

**Metodologia BEM**
Classes nomeadas conforme padrão BEM para melhor legibilidade e manutenção do código.

**BEM Flat**
Estrutura de pastas organizada pelo método BEM Flat, facilitando navegação e correção de bugs.

**Media Queries**
Implementação de media queries para responsividade em diversos dispositivos, de mobile a desktop.

**Fontes Locais**
Fontes incluídas no projeto para melhor performance.

**CSS3**
Utilizado para estilização avançada e responsiva do site.

**HTML5**
Estrutura semântica de marcação

**SEO**
Foram adicionados descrição e conteúdo no head do HTML com o intuito de melhorar o tráfego orgânico do site e torná-lo mais visível nos mecanismos de busca.

**Grid**
As seções photo-grid e places foram estilizadas com grid css, permitindo um maior controle do layout e responsividade.
